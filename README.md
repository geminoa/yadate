# yadate

Yet another `date` command.

## Usage

```sh
$ yadate -h
Yet another date command for providing more flexible ways for standard date command.

Usage:
  yadate [flags]

Flags:
  -d, --date string        display time described by STRING, not 'now'
  -h, --help               help for yadate
  -r, --reference string   display the last modification time of FILE
  -R, --rfc-email          output date and time in RFC 5322 format.  Example: Mon, 14 Aug 2006 02:34:56 -0600
  -u, --utc                print or set Coordinated Universal Time (UTC)
  ```
